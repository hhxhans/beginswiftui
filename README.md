<h4 id="Text">Text</h4>


`Text` is a view that displays one or more lines of read-only text.

Example:

```swift

Text("SwiftUI")
    .foregroundColor(.orange)
    .bold()
    .font(.system(.largeTitle))
    .fontWeight(.medium)
    .italic()
    .shadow(color: .black, radius: 1, x: 0, y: 2)

```
